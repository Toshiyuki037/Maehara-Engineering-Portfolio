## 📌 Overview

This is my personal portfolio website, built from scratch using HTML, CSS, and JavaScript.

It serves as a central place to showcase:

* engineering projects (FPGA, embedded systems, biomedical work)
* research experience
* technical skills
* resume and background

The goal was to create a fast, simple, and reliable site without relying on frameworks.

---

## ⚙️ Tech Stack

* HTML5
* CSS3
* Vanilla JavaScript
* Formspree

---

## 🚀 Features

* Responsive layout across desktop and mobile
* Project sections highlighting engineering work
* Image gallery with lightbox viewer
* Smooth scrolling and simple animations
* Optimized images for faster load times
* Contact form integration

## 🧠 Engineering Focus

While this is a frontend project, the emphasis was on:

* keeping the system lightweight (no frameworks)
* optimizing load performance
* organizing content clearly for technical presentation

The site is designed to support and present projects such as:

* FPGA-based systems
* embedded biomedical devices
* signal processing and data-driven work
